To enable GPU after reboot:
bash
# 1. Check if drivers loaded
nvidia-smi

# 2. If working, restart Ollama
sudo systemctl restart ollama

# 3. Verify GPU usage
ollama ps
# Should show "100% GPU" instead of "100% CPU"

# 4. Test speed
python3 -m assistant --no-context "hello"
# Should be 2-5 seconds instead of 10-30


If nvidia-smi doesn't work after reboot:
- Drivers may need reinstall: sudo ubuntu-drivers autoinstall
- Then reboot again
